# توضیح کلی پروژه 

پروژه‌ی سامانه امانت‌دهی کتابخانه یک سیستم کاربردی برای مدیریت کتابخانه‌ها و امانت دادن کتاب‌ها است. هدف این پروژه این است که کاربران بتوانند:
	•	کتاب‌های موجود در کتابخانه را مشاهده کنند
	•	کتاب‌ها را به فهرست امانت خود اضافه کنند
	•	اطلاعات کتاب‌ها (نام، نویسنده، ژانر، وضعیت موجودی) را مدیریت کنن
